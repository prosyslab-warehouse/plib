TUXEDO T. PENGUIN: A QUEST FOR HERRING.
by Steve and Oliver Baker

On this site you will find the three current OpenSource projects that Oliver and Steve work on:

    Tux the Penguin - A Quest for Herring:
    This is a GPL'ed game for Linux machines with 3D hardware. It runs under Windoze and MacOS too - but you do have to compile it yourself.

    Tux_AQFH has gained several top-notch awards:

    Linuxberg
    Linux Game Tome

    PLIB - A Portable Games Library/SDK
    This is the portable games library suite that was used to implement the Tux game. If you are an amateur games writer, feel free to snag a copy. It's an LGPL license.

    PLIB is also used by:


    FlightGear

    PrettyPoly Editor

    Majik3D

    PLIB has also picked up an award:

    SourceForge

    Pretty Poly Editor (PPE) - A Portable 3D Polygon-based modeller.
    This is an effort to create a modelling package suited to generating 3D models for games and other interactive applications. PPE is released under a GPL license.
